# <Sector name> — playbook

_Copy this file to `sectors/<sector>.md`, fill it in, and register it in the table in
`../sector-playbooks.md`. Answer the questions a specialist in this industry would ask —
not generic fundamentals._

**What sets the price:** the 1–2 variables this industry actually re-rates on (not P/E in the
abstract — the specific drivers).

**Catalysts to hunt (+ primary source that reveals each first):** the dated events that move
those variables.

**Valuation:** which multiple/model, on what earnings base; the mispricing signal to look for.

**Red flags:** the sector-specific ways these blow up.

**Primary sources:** where the real data lives (specific filings, trackers, standards bodies).

**Archetype / worked example:** a named case that shows the pattern.

**Variant angle:** what industry insiders track that generalist investors miss
(`../variant-perception.md`).
